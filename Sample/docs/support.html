FAQ

How do I install the add-on?

Open Google Slides, go to Extensions → Add-ons → Get add-ons. Search for "Designer's Palette" and click Install.

How do I open the sidebar?

After installation, go to Extensions → Designer's Palette → Open.

Why are some features grayed out?

Some features require specific elements to be selected. Image editing requires an image, text formatting requires a text box.

How do AI features work?

AI features process your image on secure cloud servers and return the result immediately. Original images are not stored.

What image formats are supported?

PNG, JPEG, GIF, and WebP. For best results with AI features, use high-quality source images.

Is my data secure?

Yes. Most operations happen locally. AI features use encrypted connections and do not permanently store your data.

How do I uninstall?

Go to Extensions → Add-ons → Manage add-ons. Find Designer's Palette and select Uninstall.

Known Issues

  • AI features require an internet connection
  • Very large images (over 10MB) may not be supported
  • Alignment features require at least 2 elements

Tips

  • Smaller expansion percentages yield better AI results
  • Use preview before applying changes
  • Hold Shift to select multiple objects
